
```Javascript
var users = ['Khoa', 'Nhat', 'Tam'];
```

length

```Javascript
users.length;
//output: 3
```

`array.push()`: always push the item to the end of the array

```Javascript
users.push('Miracle');
//users = ['Khoa', 'Nhat', 'Tam', 'Miracle']
```

`array.pop()`: always take off the last item of the array

```Javascript
users.push('Miracle');
//users = ['Khoa', 'Nhat', 'Tam']
```
