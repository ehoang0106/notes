
## Inline JS

```html
<body onload="alert('Hello');">
    <h1>Hello</h1>
</body>
```

## Internal JS

```html
<body>
    <h1>Hello</h1>
    <script type="text/javascript">
        alert("Hello!");
    </script>
</body>
```

## External JS

```html
<body>
    <h1>Hello</h1>
    <script src="./index.js" charset="UTF-8"></script>
</body>
```
